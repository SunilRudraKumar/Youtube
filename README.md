
# YouTube Clone

Welcome to the YouTube Clone project! This application is a comprehensive front-end clone of YouTube, focusing on advanced React techniques, complex state management with Redux, and integration with the YouTube API. Dive into the world of modern web development and experience how a large-scale application operates under the hood.

## Features

- **Advanced React Data Rendering**: Leverage React's powerful data rendering capabilities to manage video streams, user interactions, and dynamic content updates.
- **Complex State Management using Redux**: Implement Redux for global state management, ensuring smooth and predictable state transitions across the application.
- **HTTP Requests & Response Processing**: Work with HTTP requests to fetch data from the YouTube API, handling responses effectively to display content.
- **Live Debugging**: Debug in real-time, solving complex issues and ensuring a seamless user experience.
- **YouTube API Integration**: Utilize the YouTube API to fetch real video content, channel information, and more, replicating the core functionalities of YouTube.
- **Authentication**: Implement OAuth and Firebase Auth for secure user authentication and profile management.
- **Responsive Design**: Ensure the application is fully responsive and provides an optimal viewing experience across a range of devices.
- **Sass for Styling**: Use Sass, a powerful CSS extension, to write cleaner, maintainable, and reusable stylesheets.

## Getting Started

### Prerequisites

- Node.js and npm (Node Package Manager)
- Basic understanding of React, Redux, and Sass
- Google Firebase account for OAuth setup
- API key for the YouTube API

### Installation

1. Clone the repo

2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your API in `config.js`
   ```JS
   const API_KEY = 'ENTER YOUR API';
   ```
4. Setup Firebase Authentication in the Firebase Console and update the configuration in the project.

### Usage

After installation, run the application locally:

```sh
npm start
```

This will launch the YouTube Clone on your default web browser. Explore various YouTube functionalities in this clone, including searching for videos, viewing video details, and authenticating with your Google account.
